Introduction :


Python est un langage de programmation c'est-à-dire un langage informatique destiné à formuler des algorithmes et produire des programmes informatiques qui les appliquent. D'une manière similaire à une langue naturelle, un langage de programmation est composé d'un alphabet, d'un vocabulaire, de règles de grammaire, de significations, mais aussi d'un environnement de traduction censé rendre sa syntaxe compréhensible par la machine.

Python a été développé en 1989 par Guido Van Rossum, à l'Université d'Amsterdam. Guido Von Rossum était un fan de la série Monthy Python's Flying Circus d'où l'origine du nom de son langage de programmation.

Remarque : Un langage interprété (contrairement à un langage compilé) est retranscrit en langage machine au fur et à mesure de son exécution via un interprète.

Python est devenu un langage de programmation généraliste très utilisé autant dans l'industrie que dans le milieu académique et scolaire. Python est un langage open source (libre de droit et gratuit développé et utilise par une large communauté d'utilisateurs : soit 300 000 utilisateurs et plus de 500 000 téléchargements par an).

La Programmation :



I - Les différents types de données :


Il existe principalement 4 types de données dans Python :

  • Les entiers (integers) : ce sont les nombres entiers (1, 2, 3, etc)

  • Les flottants (floats) : ce sont les nombres décimaux (1.5, 2.13, 3.0, etc)

  • Les booléens (booleans) : ce sont des valeurs binaires (True / False)

  • Les chaînes de caractères (strings) : ce sont des équences de caractères ("hello", "world", etc)

Bien sûr, il existe des types de données plus compliquées tel que les listes (lists), les tuples (tuples), les ensembles (sets), les dictionnaires (dictionaries).
Certains modules de bibliothèques standarts permettent d'utiliser d'autre type de données comme les tableaux (arrays), les dates (dates), les heures (times), etc.

Il est tout-à-faitement possible de convertir des types de données entre eux.



Exemples :

Expression : >>> type(3) >>> type(2.0)
Résultat : class 'int' class 'float'

On demande d'abord avec l'instruction type(3) quel est le type de donnée de 3, et la console renvoie class 'int' pour integers, donc un nombre entier.
Puis ensuite, on demande le type de valeur de 2.0 et la console renvoie cette fois class 'float', on pourrait croire que c'est un nombre entier mais dès qu'on a un valeur décimale et ce même si elle est nulle (ici c'est 0 la valeur décimale), la valeur est considérée comme flottant.

On peut aussi changer un type de variable, par exemple si on reprend 2.0 qui est un nombre flottant et que l'on stock dans la variable x, on peut écrire :

x = 2.0
x = int(x)

La valeur de x qui était float devient donc int, x contient donc la valeur "2".
Remarque : si la valeur de x est un nombre avec une valeur décimale non-nulle, alors Python arrondira à son entier inférieur par exemple :

x = 3.7
x = int(x)

x aura donc comme valeur finale "3" et commme type de donnée "integers".



II - Les types d'opérations en Python :

Python est capable d'efféctuer plusieurs types d'opérations, on dénombre 5 grandes familles :


  1. Opérations arithmétiques :

    • Addition ( + )
    • Soustraction ( - )
    • Multiplication ( * )
    • Division ( / )
    • Division entière ou euclidienne ( // )
    • Modulo, le reste d'une division entière ( % )
    • Puissance ( ** )

  2. Opérations de comparaisons :

    • Egalité ( == )
    • Différence ( != )
    • Infériorité strictement ( < )
    • Supériorité strictement ( > )
    • Infériorité ou égalité ( <= )
    • Supériorité ou égalité ( >= )

  3. Opérations logiques :

    • ET logique ( and )
    • OU logique ( or )
    • OU EXCLUSIF logique ( ^ )
    • NON logique ( not )

  4. Opérations d'appartenances :

    • Appartenance à une séquence ( in )
    • Non-appartenance à une séquence ( not in )

  5. Opérations d'identités :

    • Identité d'objet ( is )
    • Non-identité d'objet ( is not )



Exemples :

Arithmétique : 2 + 9 Ici on fait une simple addition, la console renvoie 11
Comparaison : if x >= 20 : Ici on demande avec l'instruction conditionnel if si x (supposé un entier ou flottant) est supérieur ou égale à 20
Loqique : if x >= 20 and y != 14 : Même chose sauf que l'on cumule deux conditions dans la même instruction à savoir si x est supérieur ou égal à 20 ET que y est différent de 14
Appartenance: if x in liste : Ici on demande si x est présent dans la variable liste (qui est évidemment supposée être une liste)
Identité : x is y Ici c'est un cas un peu spécial ou l'on demande si x est y, on abordera plus en détails les opérations d'identités juste après

Dans le programme ci-dessous, on crée une liste que l'on stock dans la variable x puis une seconde, identitque, que l'on stock dans la variable y.
Puis on crée la variable z et on lui attribut la variable x.
Enfin on efféctue l'opération d'identité x is y puis x is y et on l'affiche :

x = [1, 2, 3]
y = [1, 2, 3]
z = x
print(x is y)
print(x is z)

Le programme renvoie d'abord False puis True car x et y contiennent les mêmes valeurs mais ne sont pas exactement les mêmes pour l'ordinateur, il différencie bien x et y qui sont deux variables distinctes.
Par contre la variable z qui a été créée à partir de la variable x est belle et bien identique à cette dernière pour l'ordinateur.

III - Les différents modules (ou bibliothèques) :

Sur Python, il existe différents modules (ou bibliothèques) mais d'abord, qu'es-ce qu'un module Python ?

Un module, c'est un fichier contenant des définitions et des instructions Python. Les modules sont utilisés pour organiser le code en groupant des fonctions, des classes et des variables dans un même endroit. Les modules peuvent être importés et réutilisés dans d'autres programmes Python, ce qui permet une meilleure modularité et une réutilisabilité du code.

On a par exemple les modules math détaillé ci-dessous, turtle et tkinter pour le dessin, pygame pour créer des jeux, random pour des nombres aléatoires, os pour fournir des fonctionnalités d'intéraction avec le système d'exploitation, datetime pour fournir des classes et des fonctions permettant de manipuler des dates et des heures, etc.

Module Math :

Pour commencer, voyons comment importer un module dans votre programme.
Il existe deux techniques différentes :

  • from math import *

  • import math

Dans le premier exemple, vous importez tout le module math grâce à * mais vous pouvez bien évidemment ciblez certaines classes ou fonctions, si vous connaissez leurs noms, par exemple : from math import pi, sqrt. On a importé le nombre pi et la fonction sqrt permettant de faire des racines carrées.

Dans le deuxième exemple, vous aurez besoin du préfixe 'module.' pour appelez les fonctions, par exemple : x = math.sqrt(144). Cette dernière méthode est conseillée pour éviter les conflits entre les noms des fonctions.

Remarque : Vous pouvez faire une racine carré sans le module math en tappant simplement x**0.5 mais la console renverra un nombre flottant. Pour y remédier, utiliser ce que vous avez appris plus haut pour convertir en nombre flottant en nombre entier !

Mathématiques : exp(x) In(x) sin(x) cos(x) pi √x
Python : exp(x) In(x) sin(x) cos(x) pi sqrt(x)

Remarque : L'unité des angles avec le module math de Python est le radian. Pour calculer le cosinus de 90°, il faudra donc écrire cos(pi/2).

Encore une fois, il existe plusieurs types d'instructions, on peut en dénombrer 5 grandes familles :


  • Instructions d'affectations

  • Instructions de définition de fonctions

  • Instructions d'importations

  • Instructions de sorties

Affectation : x = 5 On affecte la valeur 5 à la variable x
Conditionnelle : if x < 7.8 : On demande si x est strictement inférieur à 7.8
Définition de fonction : def function(x, y): On crée la fonction function qui prend pour argument x et y
Importation : import math On importe le module math
Sortie : print("Hello world") On fait apparaître à l'écran la chaîne de caractères Hello wolrd

Les instructions d'affectations :

Il existe différentes manières d'affecter une valeur à une variable, par exemple : x = 5, mais il existe aussi une technique dite "plus rapide" : l'affectation multiple (ou parallèle) qui consiste à affecter plusieurs valeurs à plusieurs variables avec une ligne de code :
x, y, z = 1, 2.31, True.

Ici, la variable x contient l'entier 1, la variable y contient le flottant 2.31, la variable z contient le booléen True.

On peut aussi faire des opérations plus rapidement, par exemple au lieu d'écrire x = x + 1, on peut écrire x += 1.
Cela fonctionne pour toutes les opérations arithmétiques vues plus haut.

Les instructions conditionnelles :

Il existe 2 types d'instructions conditionnelles avec chacunes quelques variantes qui seront détaillées ici.


  1. If, elif, else :

    • L'instruction if permet de vérifier une condition (littéralement "si" en français).
      Par exemple : if x != 20 :. Ici on vérifie que x est différent de 20, si la condition est validée alors les lignes suivantes indentées seront lues par l'ordinateur.

    • L'instruction elif permet de vérifier une deuxième condition si la première n'est pas validée (ce qui veut dire qu'un elif fonctionne tous le temps avec un if) : if x != 20 :
      elif x == 18 :. Ici, si x est différent de 20, alors on demande s'il est égal à 18.

    • L'instruction else permet d'exécuter des lignes de codes si aucunes des vérifications ne sont validées (évidemment il fonctionne aussi avec un if) :
      if x != 20 :
      elif x == 18 :
      else :
      Si x est différent de 20 et qu'il n'est pas égal à 18 alors l'ordinateur exécute les lignes de codes indentées sous le else.


  2. For & while :

    • La boucle for permet de répéter une opération un certain nombre de fois : for i in range(x):.
      L'ordinateur va créer une variable temporaire nommée i qui est unique à la boucle et qui va prendre successivement les valeurs de 0 à x-1 donc x itérations.

      La fonction range peut prendre jusqu'à 3 arguments, voici un tableau pour les détails (on prendra pour exemple print(i) comme ligne de code excutée dans la boucle):

      Entrée Sortie Signification
      range(5) 0, 1, 2, 3, 4 i prend toutes les valeurs par pas de 1 à partir de 0 jusqu'à 5
      range(2, 6) 2, 3, 4, 5 i prend toutes les valeurs par pas de 1 à partir de 2 jusqu'à 6
      range(1, 11, 2) 1, 3, 5, 7, 9 i prend toutes les valeurs par pas de 2 à partir de 1 jusqu'à 11

    • La boucle while permet de répéter une opération tant qu'une condition est vérifiée : while x == 1 :
      Tant que x vaut 1, la condition se répètera en boucle, sans fin.

      Important : Il faut toujours faire très attention en manipulant cette boucle car elle peut très facilement engendrer des débordements imprévus. Pensez à vérifier vous-même que la boucle s'arrêtera.
      Dans le cas où Python ne réponds plus, appuyez sur Ctrl + Alt + Suppr puis click droit sur Python et clicker sur "Fin de la tâche" Assurez-vous de bien avoir enregistrer le travail.


Remarque : Vous pouvez évidemment mettre des instructions conditonnelles dans d'autres instructions conditionnelles.
Le caractère : est présent à la fin de l'instruction pour que l'ordinateur comprenne que les lignes suivantes indentées font parties de l'instruction, c'est la syntaxe de Python.

Les instructions de définition de fonction :

Pour définir une fonction, il suffit d'écrire : def function(x): et on crée la fonction function avec x pour argument.
Cela veut dire qu'en appelant la fonction en écrivant function(5), alors l'ordinateur va exécuter les lignes indentées de la fonction et la variable x prendra la valeur 5.

On peut appeler une fonction à l'intérieur d'elle-même :

def add1(x):
-- x += 1
-- add1(x)

Cela s'appelle de la récursion et Python possède une limite qui est la "maximum recursion depth exceeded" empêchant de genre de programme de se répéter à l'infini (généralement, il s'arrête à 99 itérations).

Les instructions d'importations :

Elles permettent d'importer des modules (ou bibliothèques), elles sont détaillées plus haut.

Les instructions de sorties :

Les instructions de sorties sont des instructions permettant l'affichage comme print(x) qui affiche le contenu de la valeur x, à différencier de print("x") qui lui, affiche la chaîne de caractères "x".

Mais il existe aussi input permettant à l'utilisateur de saisir une valeur : x = input("Entrez une valeur"). L'utilisateur saisi une valeur qui est stockée dans la variable x.
